## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py --files math.csv physics.csv programming.csv --report median-coffee
```

Можно передать один или несколько файлов.

## Тесты

```bash
python3 -m pytest tests/
```
